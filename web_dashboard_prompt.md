
# Visitor Management Admin Dashboard (Flutter Web)

## Role
You are a Flutter Web frontend architect.

## Goal
Build an Admin Dashboard for society presidents to monitor visitor activity and guard operations.

## Tech Stack

- Flutter Web
- Riverpod
- REST APIs from Spring Boot backend

## Admin Dashboard Features

### Visitors Section

- View all visitor entries
- Filter by flat
- Filter by date
- Filter by status

### Residents Section

- Manage residents
- Assign flats

### Flats Section

- Generate QR codes
- View flat information

### Guards Section

- Manage security guards

### Attendance Section

- View guard attendance logs

### Roster Section

- Assign guard shifts

### Analytics Section

- Visitor count per day
- Guard attendance metrics

## UI Requirements

- Table layouts
- Filters
- Search
- Pagination
- Fast loading dashboard
